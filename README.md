# vue-tutorial

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve

After 'npm run serve' run the json-server --watch db.json to start the json server.
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
